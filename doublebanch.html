Double Branch Coupler VHF
Internal Switching Jumpers

Frequency: 20 to 175Mhz

The VBU2 is passive signal splitter that is used to branch off the main leaky feeder arterial to link into 2 side tunnels. The VBU2 is configured to provide both minimal loss as well as well matched impedanceto all potts of the unit. The branches can be configured to pass DC voltage to power the line amplifiers as well as double branch the double branch line amplifiers.

The jumpers in the branches can be configured to allow the unit to be configured in the field to:

  • Isolate or terminate the branches
  • Allow both RF as well Dc to pass through the branches
  • DC isolate the branches whilist still allowing RF to pass
